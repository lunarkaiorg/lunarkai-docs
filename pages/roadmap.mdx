# Roadmap 2025

Our vision for Lunark AI extends beyond current capabilities. Here's our strategic roadmap outlining the major developments and enhancements planned for 2025.

## January 2025: Lunark AI v0.1.0 (MVP)

- AI Assistant and Agent Framework v0.1.0
- AI Framework Memory and RAG Systems
- Basic Features (Balance check, Transfer)
- Payment & Wallet Integration

## March 2025: Lunark AI v0.5.0

- AI Framework v0.5.0
- Address Book Feature
- Token Swap Feature
- Token Bridge Feature

## June 2025: Lunark AI v1.0

- Voice Feature
- Autonomous AI Agent Task System
- Portfolio Management Feature
- Smart Contract Deployment Feature

## July 2025: Lunark AI v1.1

- Token Launch
- DAO System
- Stake System

## December 2025: Lunark AI v1.2

- Mobile App (iOS & Android)

## Continuous Improvements

- **Security**: Continuously enhancing our security measures

- **User Experience**: Improving interface and interaction methods
- **Innovation**: Implementing cutting-edge blockchain and AI technologies
