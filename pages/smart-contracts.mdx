Interact with smart contracts through natural language commands using Lunark AI. Our system simplifies complex smart contract interactions while maintaining security and transparency, making DeFi and other blockchain applications accessible to everyone.

## Understanding Smart Contracts

Lunark AI helps you understand any smart contract you're interacting with. Before each interaction, the system analyzes the contract, explains its functions, and provides clear information about potential risks and expected outcomes.

Contract Analysis Examples:

```javascript
User: "What can I do with this contract: 0x742..."
Lunark: "This appears to be Uniswap V3 Pool contract. You can:
- Add/Remove Liquidity
- Swap Tokens
- Collect Fees
- Check Pool Stats

Would you like to interact with any of these functions?"
```

```javascript
User: "Explain liquidity provision"
Lunark: "Adding liquidity to this pool:
- Token Pair: ETH/USDT
- Current TVL: $5M
- Fee Tier: 0.3%
- Price Range: Choose your range

Would you like me to help you add liquidity?"
```
